11.1 Taxonomy annotation using MicrobiotaProcess

The ggdiffcalde in MicrobiotaProcess is developed based on ggtree (Yu et al. 2017). It can be used to visualized the result of diff_analysis in MicrobiotaProcess. In addition, If you have a data frame contained hierarchical relationship, like the taxonomy annotation or KEGG levels annotation, to plot the framework, and a data frame (first column should be the labels from data frame contained hierarchical relationship, a column of factor variable, another column contains pvalue also can be provided.). It provided circular, radial, rectangular, inward_circular and slanted layout. Here, we show an example to show how to use data frame to visualize the different taxonomy tree. The data frame of this example is from the results of diff_analysis with the public datasets (Kostic et al. 2012).

library(MicrobiotaProcess)
library(ggplot2)

taxa <- read.csv("data/alltax_info.csv")
dt <- read.csv("data/difftax_tab.csv")

p <- ggdiffclade(obj=taxa,
                 nodedf=dt,
                 factorName="DIAGNOSIS",
                 layout="radial",
                 skpointsize=0.6,
                 cladetext=2,
                 linewd=0.2, # The size argument has been replace with linewd in devel version
                 taxlevel=3,
                 reduce=TRUE) + # This argument is to remove the branch of unknown taxonomy, it is provided in devel version.
     scale_fill_manual(values=c("#00AED7", "#009E73"))+
     guides(color = guide_legend(keywidth = 0.1, keyheight = 0.6,
                                 order = 3,ncol=1)) +
     theme(panel.background=element_rect(fill=NA),
           legend.position="right",
           plot.margin=margin(0,0,0,0),
           legend.spacing.y=unit(0.02, "cm"), 
           legend.title=element_text(size=7.5), # This should be adjusted with different devout.
           legend.text=element_text(size=5.5), 
           legend.box.spacing=unit(0.02,"cm")
           )

p

Figure 11.1: Different taxonomy clade.

The colors represent the relevant features enriched in the relevant classgroup. The size of point colored represent the -log10(pvalue), which means the larger point represents the higher significance. In the example, we found Fusobacterium sequences were enriched in carcinomas, and Firmicutes, Bacteroides, Clostridiales were depleted in tumors. These results were consistent with the original article (Kostic et al. 2012). In addition, we also found Campylobacter were enriched in tumors, but the relative abundance of it is lower than Fusobacterium. And the species of Campylobacter has been proven to associated with the colorectal cancer (He et al. 2019; Wu et al. 2013; Amer et al. 2017).

11.2 Visualizing phylogenetic network

ggnetworx: phylogenetic networks using ggplot2 and ggtree

library(ggplot2)
library(ggtree)
library(ggnetworx)

file <- system.file("extdata/trees/woodmouse.nxs", package = "phangorn")
Nnet <- phangorn::read.nexus.networx(file)

ggplot(Nnet, aes(x, y))  + geom_splitnet() + theme_tree()+   
    geom_tiplab2(aes(color=label), hjust=-.1)+
    scale_color_manual(values=rainbow(15)) +   
    theme(legend.position="none") +
    ggexpand(.1) + ggexpand(.1, direction=-1)
